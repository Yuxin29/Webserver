# Webserver Project
## 42 School - HTTP Server Implementation

This directory contains comprehensive documentation for the webserver project.

## Contents

- **api.html** - Complete API documentation with endpoints, methods, and examples
- **architecture.md** - System architecture and design decisions
- **testing.md** - Testing strategies and test cases

## Quick Start

Visit the main documentation page at:
http://localhost:8080/docs/api.html

## Server Ports

- **8080** - Static file server
- **8081** - CGI dynamic content server

## Key Features

1. HTTP/1.1 compliant
2. Keep-alive connections
3. Virtual hosts
4. CGI support (Python, PHP, Bash)
5. File uploads
6. Custom error pages
7. Directory listing
8. Request timeouts

# Gallery Page

This subdirectory would contain image galleries and media files.
Currently empty - ready for you to add images!

## Suggested Structure

```
gallery/
├── index.html       (gallery landing page)
├── portfolio/       (portfolio images)
├── products/        (product photos)
└── thumbnails/      (thumbnail images)
```

Try accessing http://localhost:8080/gallery with autoindex enabled to see directory listing.

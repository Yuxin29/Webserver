Welcome to the Files Directory
==============================

This directory demonstrates the autoindex feature of our webserver.

When you navigate to /files in your browser, the server will generate
a directory listing showing all files and subdirectories here.

Features:
- Supports GET, POST, and DELETE methods
- Autoindex enabled for directory browsing
- Files can be uploaded via POST requests
- Files can be deleted via DELETE requests

Try creating some files here and refreshing the browser to see the
directory listing update in real-time!

For more information, visit the home page at http://localhost:8080/

How CGI Works:

  1. Client sends request to /cgi-bin/script.py
  2. Server identifies it as CGI (matches location + extension)
  3. Http handler calls CGI executor
  4. CGI sets environment variables (REQUEST_METHOD, QUERY_STRING, etc.)
  5. CGI forks and executes the script
  6. Script output is captured and sent back to client
